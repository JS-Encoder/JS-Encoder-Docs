# 什么是 JS-Encoder?

:::tip
改文档为 JS-Encoder V4 编写，如果使用旧版请看[JS-Encoder V3 帮助文档](http://docs.lliiooiill.cn/#/)
:::

JS-Encoder 致力于成为一个方便、快速的在线 Web 实例开发、调试工具，参考了 [CodePen](https://codepen.io/)、[JSBin](https://jsbin.com/) 等优秀的在线工具，由 Vue3 和 Codemirror6 为基础构建而成，完全开源。

为了让用户能够更轻松地进行在线开发，JS-Encoder 添加了许多常用功能，本文档将为你一一介绍。

## JS-Encoder 能做什么？

- 编写代码并实时预览。
- 支持多个常用预处理语言。
- 支持使用模板，可自定义，并且支持 Vue 单文件组件的开发。
- 丰富的个性化编码设置。
- 可上传本地代码或下载 JS-Encoder 中编写的代码。
- Console 调试，支持多个常用方法以及数据结构的展示。
- 以及一些特殊功能，如 Markdown 工具栏、编码窗口视图分割等。

除此之外，还有许多功能目前正在规划中，具体请看：[TO DO LIST](https://jsbin.com/)

## 过去

最开始的 JS-Encoder V1，从功能到界面完全是模仿 [JSBin](https://jsbin.com/) 来开发的，因为当时仅仅是对这类工具的实现非常好奇。随着对这类工具的使用越来越多，便想对 V1 进行更进一步的开发。

由于该领域相关资料不算多，开发过程固然很难，磕磕绊绊但还是开发出了稳定版本 V3，V3 使用了 Vue2 和 Codemirror5 构建而成，实际上 V3 还存在一个可以在线分享、搜索代码的平台 **JS-Encoder-Online** (与 [CodePen](https://codepen.io/) 相似)，由 [@WxrLZQ](https://github.com/WxrLZQ) 负责后端开发，但这需要更多服务器会产生很多维护成本，因此不久后就下架了。

随着 Codemirror6 的逐渐稳定，2023年我开始慢慢进行 V4 的开发，截至2024年5月，V4 的功能已经基本完善。相对与 V3，新的大版本除了技术升级外，重新设计了 UI，且功能更加丰富。

## 未来

未来的 JS-Encoder 还是会具备代码分享的功能，我一直认为这是非常重要的功能，只是不再包含其他 JS-Encoder-Online 所具备的社区性功能，毕竟维护成本也是需要考虑的。
