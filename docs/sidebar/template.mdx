# 模板

在侧边工具栏点击模板选项，即可选择应用模板，包含 JS-Encoder 内置的快速模板以及用户自定义模板。

一个模板中通常包含如下配置（用户可见）：

1. 模板名
2. [预处理语言](./prep)配置以及对应的代码
3. [头部标签](./settings#头部标签)
4. 第三方库链接

同理，这也意味着在保存模板的时候也需要保存这些数据。

## 内置模板

目前的内置模板中包含几个默认模板和一个 Vue3 的单文件组件模板，选中并点击**使用改模板**按钮将其应用到编辑器中：

<img style={{width: "700px"}} src="http://img-cdn.jsencoder.cn/docs/images/inbuild-template.webp"/>

## 自定义模板

如果你想保存代码以便今后使用进行快速开发，那么可以使用自定义模板。

可以在编辑窗口内编写代码，然后点击**以当前代码创建**按钮创建模板，即可保存这些代码，在定义模板名后即可在自定义模板下看到你定义的模板：

<img style={{width: "700px"}} src="http://img-cdn.jsencoder.cn/docs/images/create-template.webp"/>

选中并点击**使用该模板**按钮将其应用到编辑器中：

<img style={{width: "700px"}} src="http://img-cdn.jsencoder.cn/docs/images/select-template.webp"/>
