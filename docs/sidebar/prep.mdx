# 预处理

在侧边工具栏点击预处理选项，即可选择预处理语言：

<img style={{width: "700px"}} src="/imgCdn/docs/images/prep.webp"/>

JS-Encoder 目前支持的预处理语言如下（取决于 Codemirror 以及社区对该语言的支持程度）：

## HTML预处理

- Markdown
- Pug

## CSS预处理

- Sass/Scss
- Less
- Stylus

## JavaScript预处理

- TypeScript
- CoffeeScript
- Babel

:::warning
如果当前使用了单文件组件模式，就无法修改预处理，你可以通过使用普通模式的模板来修改预处理。
:::
