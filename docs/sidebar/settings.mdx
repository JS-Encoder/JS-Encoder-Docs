# 编码设置

在侧边工具栏点击编码设置选项，所有可设置项：

<img style={{width: "500px"}} src="/imgCdn/docs/images/settings.png-toWebp.webp"/>

**设置完毕要记得点击确认哦！**

如果错误地进行了设置，想要重置，可以点击左下角恢复默认设置按钮：

<img style={{width: "300px"}} src="/imgCdn/docs/images/restore-settings.png-toWebp.webp"/>

## 执行

JS-Encoder 会监听编辑器内容变化，并自动执行代码，如果你想沉浸式编码，可以取消自动执行的勾选。

## 字体

可以在这里设置你的代码字体和字号，JS-Encoder 支持多种适合编码的字体。

## 头部标签

你可以在 HTML 中添加自定义头部标签，JS-Encoder 将会在编译时将它们插入到 HTML 头部 `<head></head>` 中去。
