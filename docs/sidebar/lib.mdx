# 库

在侧边工具栏点击库选项，可以搜索并选择想要引入的库，如果没有可以自行添加：

<img style={{width: "700px"}} src="http://img-cdn.jsencoder.cn/docs/images/select-lib.webp"/>

:::note
由于外网限制，使用 CDNJS 可能会导致加载失败，因此改用了国内的 BootCDN，实际上 BootCDN 里的开源项目也是同步与 CDNJS 的。
:::

## 引入顺序

JS-Encoder 按照库在列表中的顺序依次引入，如果有些库之间存在依赖关系需要更改顺序，则可以按住左边的拖动按钮来调整：

<img style={{width: "700px"}} src="http://img-cdn.jsencoder.cn/docs/images/order-lib.webp"/>
