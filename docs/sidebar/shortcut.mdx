# 快捷键

点击侧边栏的快捷键按钮查看 JS-Encoder 支持的快捷键，可以在搜索框查询快捷键：

<img style={{width: "400px"}} src="http://img-cdn.jsencoder.cn/docs/images/shortcut.png-toWebp.webp"/>

目前支持的所有快捷键如下：
